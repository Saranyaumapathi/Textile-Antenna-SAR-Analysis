📡 Implementation of Textile Antenna with SAR Measurements for Wearable Medical Applications

📌 Project Overview
This project focuses on the design and analysis of a flexible textile antenna for wearable medical communication systems. The antenna is designed to operate at 2.2 GHz with low Specific Absorption Rate (SAR), ensuring safe usage near the human body.
The design aims to achieve multi-band performance while maintaining compact size and high efficiency.

🎯 Objectives
Design a compact textile antenna for wearable applications
Achieve multi-band operation
Reduce SAR value within safety limits (1.6 W/kg standard)
Analyze radiation pattern, gain, and efficiency

🛠 Software Used
CST Microwave Studio
Antenna Simulation Tools

📊 Technical Specifications
Antenna Type: Meander Patch / Circular Monopole
Operating Frequency: 1.8 GHz – 2.2 GHz
Size: 15 × 25 × 1 mm³
Peak Gain: 3.78 dB
Radiation Efficiency: 78.2%
Low SAR for safe wearable usage

📈 Results
Return Loss (S11): -22 dB to -40 dB
Multi-band performance achieved
Stable radiation characteristics

Reduced SAR value suitable for medical applications
